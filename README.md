## Match with System

<!--eslint-skip-->

```jsonc
// .vscode/setting.json
{
  "window.autoDetectColorScheme": true,
  "workbench.preferredLightColorTheme": "Kriszu Light",
  "workbench.preferredDarkColorTheme": "Kriszu Dark",
}
```
